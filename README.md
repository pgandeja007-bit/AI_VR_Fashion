# AI–VR Fashion Intelligence Suite

Single-file Streamlit dashboard that combines:
- Classification (Logistic Regression, Random Forest, XGBoost)
- Clustering (KMeans, Agglomerative)
- Association Rule Mining (Apriori)
- Regression (Linear, Random Forest, Gradient Boosting, XGBoost)
- Dynamic Pricing Simulator
- Persona Generator
- Data upload & downloadable results

## Files
- `app.py` — main dashboard
- `requirements.txt` — Python dependencies
- `dataset.csv` — optional dataset (place at /mnt/data/AI_VR_Fashion_Survey_Synthetic_Data (1).csv for local fallback)
